# House_Price_Prediction
Built a machine learning model to predict house prices using the California Housing dataset by applying linear regression techniques and evaluating model performance.

**Goal:** Predict house prices using the California Housing dataset.

**Technologies:** Python, scikit-learn, pandas, matplotlib, seaborn

**Dataset:** California Housing Dataset (sklearn.datasets)

**Steps:**
1. Load and explore the dataset.
2. Visualize correlations and key features.
3. Split data into training and testing sets.
4. Train a Linear Regression model.
5. Predict and evaluate using R2 Score and MSE.
6. Visualize actual vs predicted house values.

**Skills Demonstrated:**
- Data Exploration
- Regression Modeling
- Model Evaluation
- Data Visualization
